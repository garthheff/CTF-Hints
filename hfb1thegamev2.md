# The Game v2

Premium room

Practice your Game Hacking skills.

Room: https://tryhackme.com/room/hfb1thegamev2

Walkthrough: https://github.com/garthheff/CTF-Writeups/blob/main/TryHackMe/hfb1thegamev2.md

## Hint 1

<details>
<summary>Hint 1</summary>

Start by identifying the downloaded files. Pay attention to whether the game has a separate resource/archive file alongside the executable.

</details>

## Hint 2

<details>
<summary>Hint 2</summary>

The game was built with a common open-source game engine. You do not need to solve this by playing the game normally.

</details>

## Hint 3

<details>
<summary>Hint 3</summary>

Look for tooling that can recover or inspect exported projects from this game engine.

</details>

## Hint 4

<details>
<summary>Hint 4</summary>

Recover the project from the packed resource file, not just the executable.

</details>

## Hint 5

<details>
<summary>Hint 5</summary>

After recovery, inspect both the decompiled scripts and the recovered assets.

</details>

## Hint 6

<details>
<summary>Hint 6</summary>

Do not focus only on text searches. Some useful information may be stored visually inside an image asset.

</details>

## Hint 7

<details>
<summary>Hint 7</summary>

One recovered asset has a name that strongly suggests it is related to the solution.

</details>

## Hint 8

<details>
<summary>Hint 8</summary>

Open the suspicious image asset directly and inspect what is displayed.

</details>
